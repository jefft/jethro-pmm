# Contributing

Jethro PMM welcomes contributions from the community.

## Reporting Bugs

Found a bug? Please report it on GitHub:

1. Go to the [Issues page](https://github.com/jethro-pmm/jethro/issues)
2. Click **New Issue**
3. Describe the bug: what you did, what you expected, what happened
4. Include your Jethro version and PHP version

## Feature Requests

Have an idea for a new feature or improvement?

1. Check the [existing issues](https://github.com/jethro-pmm/jethro/issues) to see if it's already been suggested
2. If not, create a new issue with the **Feature Request** label
3. Describe the use case and how it would work
